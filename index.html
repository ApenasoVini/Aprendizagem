REPOSITÓRIO - APRENDIZAGEM INDUSTRIAL


Lógica de Programação

Lógica de Programação é o modo de escrever um algoritmo. Que é uma sequência de instruções ou comandos realizados de maneira sistemática

Versionamento

O versionamento de software é o processo de atribuir um nome único ou uma numeração única para indicar o estado de um programa de computador.

Metodologias Ágeis

Aqui aprendemos sobre os métodos ágeis, que são um conjunto de práticas eficazes que se destinam a tornar a entrega mais rápida de produtos de alta qualidade.

APIs

Aqui aprendemos sobre a forma de comunicação entre sistemas. Ou seja, a integração entre dois sistemas, em que um deles fornece informações e serviços que podem ser utilizados pelo outro.

Banco de Dados

Aqui aprendemos sobre Banco de Dados, uma coleção organizada de informações, ou dados, projetada para permitir o armazenamento, recuperação, gerenciamento e atualização dessas informações. O objetivo principal é fornecer um meio de gerenciar grandes quantidades de informações de maneira mais eficiente.
